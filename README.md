# О проекте

Интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки. Страница адаптивная и подстраивается под большинство популярных устройств.

# Стэк 

- React
- JS
- CSS
- HTML
- Webpack
- БЭМ
- Figma
- Git
